# ivanpriporov-website

My own portfolio website

## О проекте

Этот проект — одностраничный сайт-портфолио, который можно использовать как личную визитку в интернете.  
На странице есть основные блоки: приветствие, информация обо мне, навыки, проекты и контакты.

## Стек

- HTML
- CSS

## Возможности

- Адаптивная верстка под разные экраны.
- Плавная навигация по секциям страницы.
- Чистая и минималистичная структура.

## Структура проекта

```bash
project-root/
├── index.html
├── src/
│   ├──css/
│   │   └── style.css
│   └── images/
│        └── ...
└── README.md
```

## Как запустить

1. Склонируй репозиторий:

```bash
git clone https://github.com/ewanpriporov/ivanpriporov-website.git
```

2. Перейди в папку проекта:

```bash
cd ivanpriporov-website
```

3. Открой файл `index.html` в браузере.
